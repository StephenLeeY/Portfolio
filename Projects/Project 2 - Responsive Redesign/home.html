Problems with Old Design

Learnability

  • Hidden/obscure interactive elements
  • Multiple navigation bars in different styles
  • Navigation buttons hidden in slideshow effect

    • Usability

  • Menu and Order Online sections are separate
  • Static elements / Lacks reactive UI

    • Memorability

  • Sign in feature hidden in “Order Online” section
  • Lack of clear visual hierarchy
  • Repeated information across separate sections

    • Accessibility

    El Patron’s has significant accessibility issues, especially for the visually impaired. Screen readers have significant difficulty parsing through the second half of the website, often repeating words multiple times and not making any sense at all. Many linked images lack alternative text, which only contributes to this problem. Furthermore, the website lacks a language tag, meaning that screen readers will be unable to translate the site to non-english languages.
